## 💡 Why Data Cleaning?

Real-world datasets are rarely perfect. Before performing analysis or building machine learning models, data must be cleaned to ensure:

* Better accuracy
* Reliable analysis
* Improved visualization
* Better model performance
* Reduced errors

## 📸 Sample Operations

```python
# Remove duplicate rows
df.drop_duplicates()

# Drop unwanted columns
df.drop(columns=["Column_Name"])

# Fill missing values
df.fillna(0)

# Convert data type
df["Phone_Number"] = df["Phone_Number"].astype(str)
```
